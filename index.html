Curso de HTML5 com CSS3

A HTML5 é uma linguagem de marcação para a criação de sites. Atrualmente na versão 5, ela usa a semantica dos elementos para criar o conteúdo de uma pagina Web. Com as CSS3, é possível criar páginas com aparência moderna e responsiva.

Curso de JavaScript

O JS é uma linguagem de programação que permite criar interatividade em páginas web. Com o JavaScript, é possível criar animações, validar formulários, manipular elementos da página e muito mais. Os codigos em JS são executados dentro do navegador e dão acesso aos elementos HTML por meio do DOM.
